#  Find santa game🎅🏻
**당근게임을 업그레이드한 산타찾기 게임입니다**
1. 재생버튼 누를시 타이머 스타트/ 산타1,2와 눈사람은 랜덤 위치에 포지션
2. 찾아야할 산타 이미지 모달로 보여줌
3. 산타 클릭시 스코어의 숫자 줄어듬
4. 산타2와 눈사람 클릭시 게임오버
5. 산타1을 전부 클릭시 you win모달 뜸
6. replay 누를시 처음으로 리셋
7. 산타 1의 움직임 추가
8. 산타2와 눈사람의 애니메이션 추가
